# WooCommerce Product Profit Calculator

A professional WooCommerce plugin that allows store owners to track product cost prices and automatically calculate profit, markup and margin.

## Features

- Cost price field
- Live profit calculation
- Markup percentage
- Profit margin
- Variable product support
- HPOS compatible
- Translation ready

## Requirements

- WordPress 4.0+
- WooCommerce 2.6+
- PHP 5.6+
